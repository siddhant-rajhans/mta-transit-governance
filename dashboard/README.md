# Dashboard

The Power BI file (`MTA_Transit_Dashboard.pbix`) and screenshots live
here.

If you don't have Power BI Desktop, the screenshots in `screenshots/`
should be enough to see what's on each of the three pages.

To open the .pbix: install Power BI Desktop (free, Windows only), then
double-click. The data sources are the CSVs under `../data/clean/`. If
those aren't there yet, run the scripts in `../scripts/` first.
